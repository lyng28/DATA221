# DATA221 Assignments

Each question is in its own file.

- `DATA221_Assignment1_Question1.py`: Multiplies consecutive integers starting at 1 until the product exceeds a threshold.
- `DATA221_Assignment1_Question2.py`: Builds a nested dictionary from a list of strings.
- `DATA221_Assignment1_Question3.py`: Computes x^y for pairs and skips pairs with negative exponents.
- `DATA221_Assignment1_Question4.py`: Sorts random values and finds indices where values are >= x.
- `DATA221_Assignment1_Question5.py`: Validates circle radii and returns the coverage percentage of smaller area over larger area.
- `DATA221_Assignment1_Question6.py`: Return a sorted dictionary of percentages of values <= each unique key.
- `DATA221_Assignment1_Question7.py`: Converts seconds since midnight into hours, minutes, seconds, and AM/PM format.
- `DATA221_Assignment1_Question8.py`: Creates a pandas DataFrame and adds a computed column.
